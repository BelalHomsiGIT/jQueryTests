By default, all HTML elements have a static position, and cannot be moved. To manipulate the position, remember to first set the CSS position property of the element to relative, fixed, or absolute!

Animate Effect

Is it possible to manipulate ALL CSS properties with the animate() method? Yes, almost! However, there is one important thing to remember: all property names must be camel-cased when used with the animate() method: You will need to write paddingLeft instead of padding-left, marginRight, instead of margin-right, and so on.
If you want to animate color, you need to download the Color Animations plugin from jQuery.com.
